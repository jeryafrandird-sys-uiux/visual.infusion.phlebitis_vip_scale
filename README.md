# Visual.Infusion.Phlebitis_VIP_Scale
Aplikasi web ini adalah prototipe digital untuk pemantauan area infus menggunakan Visual Infusion Phlebitis (VIP) Scale. Dirancang untuk membantu perawat melakukan deteksi dini tanda flebitis secara akurat melalui antarmuka responsif. Fitur utama mencakup penilaian skor VIP otomatis dan rekomendasi tindakan klinis untuk keselamatan pasien.
